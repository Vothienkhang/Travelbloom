# TravelBloom

A small multi‑page travel recommendation website built with **HTML/CSS/JS**.

## Pages
- `index.html` – Home (navbar, hero intro, search, recommendations + Beach/Temple/Country sections)
- `about.html` – About Us
- `contact.html` – Contact form (demo client‑side only)

## Run locally
Open `index.html` in a browser, or use VS Code Live Server.

## Deploy to GitHub Pages
1. Create a public repo, push files.
2. Settings → Pages → Deploy from a branch → `main` (root).
3. Wait for the live URL: `https://<your-user>.github.io/<repo>/`.
